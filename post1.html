Web Scraping and Job Postings

This is my solo assignment about web scraping and natural language processing.

I scraped the data science job details with the Beautifulsoup package from the job searching website Indeed. I scraped 3 categories of job: data scientist, data analyst and BI analyst.

Then I created the dataframe and converted the object features to numerical features with NLP techniques (TF/IDF model). Thereafter I completed the feature importance analysis of job category and salary classifications.

All of algorithms (decision tree, random forest, and garident boosting) have brought good accuracy scores and I have visualized the feature importance.
